Instalar paquete django:

pip install Django==3.2.8

Instalar paquetes para correr la base de datos:

pip install PyMySQL
pip install Pillow

Ingresar al sistema en consola:

cd emmawood

Migrar del sistema la base de datos:

py manage.py makemigrations
py manage.py migrate

Correr el servidor:

py manage.py runserver
